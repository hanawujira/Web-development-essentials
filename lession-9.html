• color: Sets the color of a text using RGB, hex, or keyword values.
  • line-height: Sets the distance between lines using normal, number, length, or percentage values.
  • letter-spacing: Increases or decreases the space between characters using normal or number values.
  • text-align: Aligns the text in an element using left, right, center, or justify values.
  • text-decoration: Adds decoration to text using none, underline, overline, or line-through values.
  • font-family: Specifies the font family for the text. The font family can be a single font name or a list of font names, separated by commas. If the browser does not have the specified font installed, it will use a fallback font.
  • font-size: Specifies the size of the text. The font size can be specified in pixels (px), points (pt), or ems (em).
  • font-style: Specifies the style of the text, such as normal, italic, or oblique.
  • font-weight: Specifies the weight of the text, such as normal, bold, or lighter.
